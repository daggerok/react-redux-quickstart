# react-redux-quickstart [![build](https://travis-ci.org/daggerok/angular4.svg?branch=webpack)](https://travis-ci.org/daggerok/angular4)

stack:
- react
- redux
- react-redux
- webpack 2

develop

```bash
npm i -g yarn
yarn install
yarn build
yarn serve
yarn start
```

update

```bash
yarn upgrade
# or
npm i -g npm-check-updates
ncu -u -x
```
